## Library Content

The core library contains several packages that can be built. Below is a list of the available packages along with a brief description of their contents.

| Package Name | Description |
|--------------|-------------|
| [`ocd_aerodynamics_model_base_cpp`](./ocd_aerodynamics_model_base_cpp/README.md) | Base class for aerodynamics models |
| [`ocd_aerodynamics_models_cpp`](./ocd_aerodynamics_models_cpp/README.md) | A package containing different implementations of aerodynamics models |
| [`ocd_drivetrain_fx_cpp`](./ocd_drivetrain_fx_cpp/README.md) | Drivetrain Equations for a Rear Wheel Driven Car with LSD and Longitudinal Force as Input |
| [`ocd_drivetrain_model_base_cpp`](./ocd_drivetrain_model_base_cpp/README.md) | Base classes and concepts for drivetrain models in C++ |
| [`ocd_drivetrain_rwd_lsd_cpp`](./ocd_drivetrain_rwd_lsd_cpp/README.md) | Drivetrain Equations for a Rear Wheel Driven Car with LSD |
| [`ocd_drivetrain_wheel_torque_cpp`](./ocd_drivetrain_wheel_torque_cpp/README.md) | A drivetrain model using direct torque control implemented in C++. |
| [`ocd_helpers_cpp`](./ocd_helpers_cpp/README.md) | A package providing helper functions for simulation in C++ |
| [`ocd_steering_actuator_model_base_cpp`](./ocd_steering_actuator_model_base_cpp/README.md) | Base class for a steering actuator model |
| [`ocd_steering_actuator_pt1_cpp`](./ocd_steering_actuator_pt1_cpp/README.md) | A simple steering actuator model with pt1 dynamics |
| [`ocd_tire_model_base_cpp`](./ocd_tire_model_base_cpp/README.md) | Base class for tire models |
| [`ocd_tire_models_cpp`](./ocd_tire_models_cpp/README.md) | A package containing different implementations of tire models |
| [`ocd_types_cpp`](./ocd_types_cpp/README.md) | A package providing type definitions for simulation in C++ |
| [`ocd_vehicle_dynamics_double_track_cpp`](./ocd_vehicle_dynamics_double_track_cpp/README.md) | A double track vehicle dynamics model implemented in C++ |
| [`ocd_vehicle_dynamics_model_base_cpp`](./ocd_vehicle_dynamics_model_base_cpp/README.md) | Base class for a vehicle dynamics model |
| [`ocd_vehicle_dynamics_single_track_cpp`](./ocd_vehicle_dynamics_single_track_cpp/README.md) | A single track vehicle dynamics model implemented in C++ |
| [`ocd_vehicle_model_base_cpp`](./ocd_vehicle_model_base_cpp/README.md) | Base class for vehicle models |
| [`ocd_vehicle_model_cpp`](./ocd_vehicle_model_cpp/README.md) | A generic vehicle model implementation in C++ which can be composed from a drivetrain, steering actuator and a dynamics model |
